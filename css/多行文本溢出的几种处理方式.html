多行文本溢出的几种处理方式

在CSS中，实现单行文本溢出时显示省略号是一件非常容易的事，如下：

{
    overflow: hidden;
    white-space: no-wrap;
    text-overflow: ellipsis;
}

当时当在多行文本溢出时要达到同样的效果，我们该如何做呢？

多行文本溢出显示省略号在国外一般有个专门的术语叫“行夹（line clamping）”，有很多方法可以达到这样的效果，但是没有一种是包打天下的，都有各式各样的兼容性问题。

下面我们来逐个介绍。

一、WebKit Flexbox 方式

David DeSandro在早期介绍了这一属性，我们来引用它的话：

-webkit-line-clamp是一个不受支持的WebKit属性，它限制块元素中显示的文本行数。 为了达到这个效果，它需要与其他一些奇特的WebKit属性组合在一起。

假设我们有个p元素，我们需要使用这三个属性：

.line-clamp {
  overflow: hidden;
  display: -webkit-box;
  -webkit-line-clamp: 3;
  -webkit-box-orient: vertical;  
}

如下：

一方面，这真是太棒了，正是我们需要的。CSS在这个地方是相当完美的。它就是应该这么简单。 另一方面它有很多缺点：

  1. 只支持webkit（庆幸大多数移动端浏览器是webkit的）
  2. 必须要设置display: -webkit-box;属性
  3. 段落如果有一些padding，那它就没法正常工作了。

二、Fade Out 方式

这种方式的核心就是以可预测的方式设包含块的高度。假设你的line-height属性是line-height： 1.2em，然后我们只想它最多显示3行，那么，我们需要将包含块的height设置为height: 3.6em;然后我们接着用overflow: hidden;隐藏剩下的内容。如下：

可以看到，超出部分缺失被影藏起来了，但是，像这样可能有点尴尬。 理想情况下，我们会添加省略号，但我们无法可靠地定位它们。所以省略号并不好加。 大多数情况下，我们都是将最后一行进行淡出来达到某种视绝效果。但这种效果看起来肯定没有省略号好。

为了淡出最后一行，我们创建一个框（通常是伪元素），并在顶部覆盖透明背景颜色渐变。 将它设置的与容器几乎一样宽，以免最后一行实际文本很短。 因为我们知道行高，所以我们可以将伪元素框设置为一行高。

也就是下面这段代码：

.fade-out {
    position: relative;
    line-height: 1.2em;
    height: 3.6em;
    overflow: hidden;
}
.fade-out:after {
  content: "";
  text-align: right;
  position: absolute;
  bottom: 0;
  right: 0;
  width: 70%; // 将它设置的与容器几乎一样宽，以免最后一行实际文本很短。
  height: 1.2em;
  background: linear-gradient(to right, rgba(255, 255, 255, 0), rgba(255, 255, 255, 1) 50%);
}

我们来看看效果：

如果想将渐变色的stop设置在靠右，我们可以把上面linear-gradient的50%改成90。有的人还想将content: "";改为content: "...";达到省略号的效果，如下：

但是这个其实不太好加，因为你根本不能确定最后一行文本到底有多长，有时候比较短，更本就没有溢出的时候，加上省略号会显得非常的突兀。

三、Opera浏览器 Overflow方法

和WebKit一样，Opera有自己的方式来处理这个问题。 他们在你想要的行上应用省略号。 当然，这个只能用在Opera上，考虑到Opera的市场份额，所以也不是特别有用。

.opera-way {
  height: 3.6em; /* exactly three lines */
  line-height: 1.2em;
  text-overflow: -o-ellipsis-lastline;
}

但是呢，笔者在现在的Opera上体验， 发现这个属性已经不能用了，所以这个方法还是不要再用了。

四、Clamp.js方式

这是一种使用JS实现的方式，使用起来非常简单，可查看Github, 用法：

var module = document.getElementById("clampjs");
$clamp(module, {clamp: 3});

五、ftellipsis方式

ftellipsis.js是FT实验室开发的JS插件，用于达到同样的效果，使用方法：

// https://github.com/ftlabs/ftellipsis
var element = document.getElementById('ftellipsis');
var ellipsis = new Ellipsis(element);

ellipsis.calc();
ellipsis.set();

效果同上。
