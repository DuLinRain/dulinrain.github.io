CSS居中详解

一、概述

CSS居中是一个老生常谈的问题，不管是面试中还是实际应用中，遇到的都很多，对于不同类型的元素，其水平、垂直、水平垂直居中的方法也不尽相同，直接泛泛而论总是容易忘记，我们今天进行系统的总结一下。

二、水平居中

当谈到水平居中的时候，我们首先要从一下几个方面问自己：

  1. 它是inline元素(比如文本或链接)或者有dispaly: inline/inline-*属性吗？
  2. 它是块级元素吗？
  3. 是需要居中单个块级元素还是多个？

当我们从这三个方面问我们自己的时候，答案也就一一浮现出来了。下面我们分别讨论。

2.1、对inline元素或具有dispaly: inline/inline-*属性的元素

只需要在父块级元素中设置text-align: center;即可。这适用于inline, inline-block, inline-table, inline-flex, 等元素。

举例：

<header>
  This text is centered.
</header>

<nav role='navigation'>
  <a href="#0">One</a>
  <a href="#0">Two</a>
  <a href="#0">Three</a>
  <a href="#0">Four</a>
</nav>  

body {
  background: #f06d06;
}

header, nav {
  text-align: center;
  background: white;
  margin: 20px 0;
  padding: 10px;
}

nav a {
  text-decoration: none;
  background: #333;
  border-radius: 5px;
  color: white;
  padding: 3px 8px;
}

结果：

2.2、对块级元素

对于有固定宽度的块级元素，只需要设置margin: 0 auto；即可，对于无固定宽度的元素，它本身会铺满整个宽度，所以就没有居中不居中的问题了。

举例：

<main>
  <div class="center">
    I'm a block level element and am centered.
  </div>
</main>

body {
  background: #f06d06;
}

main {
  background: white;
  margin: 20px 0;
  padding: 10px;
}

.center {
  margin: 0 auto;
  width: 200px;
  background: black;
  padding: 20px;
  color: white;
}

结果：

2.3、是居中一个还是多个块级元素？

对于单纯的在一行居中一个块级元素，按2.2方式来处理就OK了，但是如果想在一行水平居中多个块级元素呢？

有两种方法可以做到：

  1. 将它们设置成inline-block元素，然后使用text-align: center;
  2. 使用flex布局。

例子：

<main class="inline-block-center">
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row.
  </div>
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row. I have more content in me than my siblings do.
  </div>
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row.
  </div>
</main>

<main class="flex-center">
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row.
  </div>
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row. I have more content in me than my siblings do.
  </div>
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row.
  </div>
</main>

body {
  background: #f06d06;
  font-size: 80%;
}

main {
  background: white;
  margin: 20px 0;
  padding: 10px;
}

main div {
  background: black;
  color: white;
  padding: 15px;
  max-width: 125px;
  margin: 5px;
}

.inline-block-center {
  text-align: center;
}
.inline-block-center div {
  display: inline-block;
  text-align: left;
}

.flex-center {
  display: flex;
  justify-content: center;
}

结果：

如果对于多个块级元素你想要的是下面这种羊肉串效果，你直接用2.2中的margin: 0 auto；即可：

<main>
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row.
  </div>
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row. I have more content in me than my siblings do.
  </div>
  <div>
    I'm an element that is block-like with my siblings and we're centered in a row.
  </div>
</main>

body {
  background: #f06d06;
  font-size: 80%;
}

main {
  background: white;
  margin: 20px 0;
  padding: 10px;
}

main div {
  background: black;
  margin: 0 auto;
  color: white;
  padding: 15px;
  margin: 5px auto;
}

main div:nth-child(1) {
  width: 200px;
}
main div:nth-child(2) {
  width: 400px;
}
main div:nth-child(3) {
  width: 125px;
}

三、垂直居中

垂直居中时也需要考虑两种情况：

  1. 它是inline元素(比如文本或链接)或者有dispaly: inline/inline-*属性吗？
  2. 它是块级元素吗？

下面我们来分别看看。

3.1、对inline元素(比如文本或链接)或者有dispaly: inline/inline-*属性

这种情况其实又可以分为三种情况分别对待：

  1. 它是单行吗？
  2. 它是多行吗？

3.1.1 对单行元素

单行元素的话，实际上我们常常可以设置相同的上、下padding。

例子：

<main>
  <a href="#0">We're</a>
  <a href="#0">Centered</a>
  <a href="#0">Bits of</a>
  <a href="#0">Text</a>
</main>

body {
  background: #f06d06;
  font-size: 80%;
}

main {
  background: white;
  margin: 20px 0;
  padding: 50px;
}

main a {
  background: black;
  color: white;
  padding: 40px 30px;
  text-decoration: none;
}

结果：

如果这样不起作用的话，我们还可以考虑设置line-height和height相等。

例子：

<main>
  <div>
    I'm a centered line.
  </div>
</main>

body {
  background: #f06d06;
  font-size: 80%;
}

main {
  background: white;
  margin: 20px 0;
  padding: 40px;
}

main div {
  background: black;
  color: white;
  height: 100px;
  line-height: 100px;
  padding: 20px;
  width: 50%;
  white-space: nowrap;
}

结果：

3.1.2 对多行元素

对多行元素，又可以分成5种方法：

  1. 首先我们仍然可以采用相同的上下padding实现居中效果
  2. 如果这种方法不起作用的话，我们可以考虑直接将元素放在table的一个单元格中
  3. 或者设置元素父元素的display: table-cell；以及vertical-align: middle;属性。
  4. 如果还是不行，可以考虑使用flexbox。
  5. 使用伪元素

例子：

直接放在table单元格中：

<table>
  <tr>
    <td>
      I'm vertically centered multiple lines of text in a real table cell.
    </td>
  </tr>
</table>

body {
  background: #f06d06;
  font-size: 80%;
}

table {
  background: white;
  width: 240px;
  border-collapse: separate;
  margin: 20px;
  height: 250px;
}

table td {
  background: black;
  color: white;
  padding: 20px;
  border: 10px solid white;
  /* default is vertical-align: middle; */
}

结果：

设置父元素的display: table-cell；以及vertical-align: middle;属性：

<div class="center-table">
  <p>I'm vertically centered multiple lines of text in a CSS-created table layout.</p>
</div>

.center-table {
  display: table;
  height: 250px;
  background: white;
  width: 240px;
  margin: 20px;
}
.center-table p {
  display: table-cell;
  margin: 0;
  background: black;
  color: white;
  padding: 20px;
  border: 10px solid white;
  vertical-align: middle;
}

结果：

使用flex布局例子：

<div class="flex-center">
  <p>I'm vertically centered multiple lines of text in a flexbox container.</p>
</div>
body {
  background: #f06d06;
  font-size: 80%;
}

div {
  background: white;
  width: 240px;
  margin: 20px;
}

.flex-center {
  background: black;
  color: white;
  border: 10px solid white;
  display: flex;
  flex-direction: column;
  justify-content: center;
  height: 200px;
  resize: vertical;
  overflow: auto;
}
.flex-center p {
  margin: 0;
  padding: 20px;
}

注意：上面那种方法需要父元素有自己的固定高度，否则可能不会起作用。

如果以上方法都失效的话，可以采用最后一种伪元素的方法：

<div class="ghost-center">
  <p>I'm vertically centered multiple lines of text in a container. Centered with a ghost pseudo element</p>
</div>

body {
  background: #f06d06;
  font-size: 80%;
}

div {
  background: white;
  width: 240px;
  height: 200px;
  margin: 20px;
  color: white;
  resize: vertical;
  overflow: auto;
  padding: 20px;
}

.ghost-center {
  position: relative;
}
.ghost-center::before {
  content: " ";
  display: inline-block;
  height: 100%;
  width: 1%;
  vertical-align: middle;
}
.ghost-center p {
  display: inline-block;
  vertical-align: middle;
  width: 190px;
  margin: 0;
  padding: 20px;
  background: black;
}

结果：

3.2、对块级元素

这种情况也可以分成三种小情况区别对待：

  1. 元素高度已知
  2. 元素高度未知
  3. 可否用flexbox

我们分别来探讨。

3.2.1 元素高度已知

这种情况下使用绝对定位+ 负margin即可。

举例：

<main>

  <div>
     I'm a block-level element with a fixed height, centered vertically within my parent.
  </div>

</main>

body {
  background: #f06d06;
  font-size: 80%;
}

main {
  background: white;
  height: 300px;
  margin: 20px;
  width: 300px;
  position: relative;
  resize: vertical;
  overflow: auto;
}

main div {
  position: absolute;
  top: 50%;
  left: 20px;
  right: 20px;
  height: 100px;
  margin-top: -70px;
  background: black;
  color: white;
  padding: 20px;
}

结果：

3.2.2 元素高度未知

这种情况下使用transform: translateY(-50%)即可。

<main>

  <div>
     I'm a block-level element with an unknown height, centered vertically within my parent.
  </div>

</main>

body {
  background: #f06d06;
  font-size: 80%;
}

main {
  background: white;
  height: 300px;
  margin: 20px;
  width: 300px;
  position: relative;
  resize: vertical;
  overflow: auto;
}

main div {
  position: absolute;
  top: 50%;
  left: 20px;
  right: 20px;
  background: black;
  color: white;
  padding: 20px;
  transform: translateY(-50%);
  resize: vertical;
  overflow: auto;
}

结果：

3.2.3 可否使用flexbox

如果浏览器支持flexbox的话那就更简单了，使用flexbox居中即可。

例子：

<main>

  <div>
     I'm a block-level element with an unknown height, centered vertically within my parent.
  </div>

</main>

body {
  background: #f06d06;
  font-size: 80%;
}

main {
  background: white;
  height: 300px;
  width: 200px;
  padding: 20px;
  margin: 20px;
  display: flex;
  flex-direction: column;
  justify-content: center;
  resize: vertical;
  overflow: auto;
}

main div {
  background: black;
  color: white;
  padding: 20px;
  resize: vertical;
  overflow: auto;
}

结果：

四、水平垂直居中

水平垂直居中实际上也分几种情况：

  1. 元素宽度和高度已知
  2. 元素宽度和高度未知
  3. 可以使用flexbox吗
  4. 可以使用grid吗

我们分开讨论。

4.1、元素宽度和高度已知

这种情况也是使用position: absolute; + 负margin实现。

例子：

<main>

  <div>
     I'm a block-level element a fixed height and width, centered vertically within my parent.
  </div>

</main>

body {
  background: #f06d06;
  font-size: 80%;
  padding: 20px;
}

main {
  position: relative;
  background: white;
  height: 200px;
  width: 60%;
  margin: 0 auto;
  padding: 20px;
  resize: both;
  overflow: auto;
}

结果：

4.2、元素宽度和高度未知

这种情况下使用transform: translate(-50%, -50%)实现。

例子：

<main>

  <div>
     I'm a block-level element of an unknown height and width, centered vertically within my parent.
  </div>

</main>

body {
  background: #f06d06;
  font-size: 80%;
  padding: 20px;
}

main {
  position: relative;
  background: white;
  height: 200px;
  width: 60%;
  margin: 0 auto;
  padding: 20px;
  resize: both;
  overflow: auto;
}

main div {
  background: black;
  color: white;
  width: 50%;
  transform: translate(-50%, -50%);
  position: absolute;
  top: 50%;
  left: 50%;
  padding: 20px;
  resize: both;
  overflow: auto;
}

结果同上。

4.3、可以使用flexbox

这种情况使用flex布局即可：

例子：

<main>

  <div>
     I'm a block-level-ish element of an unknown width and height, centered vertically within my parent.
  </div>

</main>

body {
  background: #f06d06;
  font-size: 80%;
  padding: 20px;
}

main {
  background: white;
  height: 200px;
  width: 60%;
  margin: 0 auto;
  padding: 20px;
  display: flex;
  justify-content: center;
  align-items: center;
  resize: both;
  overflow: auto;
}

main div {
  background: black;
  color: white;
  width: 50%;
  padding: 20px;
  resize: both;
  overflow: auto;
}

结果同上。

4.4、可以使用grid布局

如果可以使用grid布局的话，也相当简单：

<span>
  I'm centered!
</span>

body, html {
  height: 100%;
  display: grid;
}
span {
  margin: auto;
}
