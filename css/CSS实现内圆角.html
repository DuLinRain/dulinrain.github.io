CSS实现内圆角

CSS实现圆角我们一定都已经很熟悉了，但是对于如下图所示的内圆角我们该如何实现呢？

很显然，我们可以用两个元素来实现，但是这样又会导致出现无意义的元素，那么有没有更好的方法呢？答案是肯定的。我们可以使用outline+box-shadow来实现，当然我们也可以使用伪元素实现。

1、使用outline + box-shadow 实现内圆角

想必看过CSS模拟多边框这篇文章的一定会知道用box-shadow来模拟任意多边框，用outline模拟2个边框。也应该记得我们提过box-shadow会跟随圆角，而outline不会跟随圆角。我们再回顾一下有圆角的时候outline的样子：

css代码：

#inner-round{
    width: 200px;
    height: 100px;
    background-color: tan;
    border-radius: .8em;
    margin: 40px auto;
    padding: 1em;
    outline: .6em solid #655;
}

效果：

可以看到，上面的代码几乎就实现了我们的预期效果，除了在元素和outline之间的四个角上存在空白，那么我们自然会想什么东西可以将它去掉呢？

如果我们能用和outline一样颜色的东西盖住空白角不就OK了吗，而我们前面提到过box-shadow会跟随圆角，所以我们可以用box-shadow实现：

CSS代码：

#inner-round{
    width: 200px;
    height: 100px;
    background-color: tan;
    border-radius: .8em;
    margin: 40px auto;
    padding: 1em;
    box-shadow: 0 0 0 .5em #655;
    outline: .6em solid #655;
}

效果：

其实我们用outline去盖住空白角时可以指定其他的outline宽度，那么最小可以指定多小就可以了呢？我们可以从下面这个示意图看出：

这意味着，我们可以指定的最小outline宽度是： 圆角半径 * （根号2 - 1）。同理outline的宽度也不应该超过box-shadow的宽度。

2、使用伪元素 + background-color + box-shadow实现

CSS代码：

#inner-round{
    width: 200px;
    height: 100px;
    background-color: tan;
    border-radius: .8em;
    margin: 40px auto;
    padding: 1em;
    position: relative;
}
#inner-round::before {
    position: absolute;
    content: '';
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    background: #655;
    box-shadow: 0 0 0 .5em #655;
    z-index: -1;
}

效果同上。

3、使用伪元素 + background-color + outline实现

CSS代码：

#inner-round{
    width: 200px;
    height: 100px;
    background-color: tan;
    border-radius: .8em;
    margin: 40px auto;
    padding: 1em;
    position: relative;
}
#inner-round::before {
    position: absolute;
    content: '';
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    background: #655;
    outline: .5em solid #655;
    z-index: -1;
}

效果同上。
