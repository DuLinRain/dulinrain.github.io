CSS实现平行四边形效果

平行四边形是网页设计中一种非常常见的形状，既用在普通的元素也用于导航栏场景，如下所示：

大部分人首先想到的就是用css形变来实现，如下：

transform: skewX(-45deg);

但是这样实现的效果如下图所示：

这样会导致一个明显的问题是文本也倾斜了，这通常不是我们想要的效果。那么要想保持元素倾斜而文本不倾斜有什么方法呢？我们可以使用嵌套元素和伪元素两种方法来实现。

1、嵌套元素实现

嵌套元素的实现思路非常简单，我们采用两层结构，对外层元素skew(-45deg)，对内层元素skew(45deg)，这样内层元素的文本就摆正了。

css代码：

#parrell {
    width: 100px;
    height: 36px;
    margin: 50px auto;
    background-color: yellow;
    transform: skew(-45deg);
}
#parrell div {
    line-height: 36px;
    text-align: center;
    transform: skew(45deg);
}

效果：

很明显，这样相当于会有无意义的元素，这通常是不可取的方法。更合适的方式是使用伪元素。

2、伪元素实现

伪元素实现的思路也很明显，我们对伪元素施加skew(-45deg)即可：

CSS代码

#parrell {
    width: 100px;
    height: 36px;
    margin: 50px auto;
    line-height: 36px;
    text-align: center;
    position: relative;
}
#parrell::after{
    position: absolute;
    content: '';
    left: 0;
    top: 0;
    right: 0;
    bottom: 0;
    background-color: yellow;
    transform: skew(-45deg);
    z-index: -1;
}

效果同上

这种方法不仅适用于skew，对于任何其他“想对元素形变，但保持内容不形变”的场合都能适用。
