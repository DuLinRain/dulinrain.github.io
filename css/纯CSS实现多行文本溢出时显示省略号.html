纯CSS实现多行文本溢出时显示省略号

我们在多行文本溢出的几种处理方式这篇文章中总结了实现多行文本溢出时的多种处理方法，其中提到了WebKit Flexbox方式和fadeOut两种CSS方式以及几种JS方式。其中，这里提到的2中CSS处理方式都有自己的缺点，JS方式更是不推荐的。今天我们来学习一种新的纯CSS的实现方式，这种实现方式需要较大的想象力才能做到。我们将一步一步地教你实现。

一、step1: 想法

实现多行文本溢出显示省略号的难处在于怎么判断多行文本是否溢出了。显然，在没有JS介入的情况下，我们很难做到。但是我们可以从CSS的float特性入手来实现。我们知道，当元素设置了float属性之后，它们会脱离文档流。float元素不会互相覆盖，会根据它之前的元素来调整自己的位置。比如下面这段代码：

.wrap {
    width: 400px; height: 200px;
    margin: 20px 20px 50px;
    border: 5px solid #AAA;
    line-height: 25px;
}
.prop, .main, .end {
    opacity: 0.7;
}
.prop {
    float: left;
    width: 100px; height: 200px;
    background: #FAF; 
}
.main {
    float: right;
    width: 300px;
    background: #AFF; 
}
.end {
    float: right;
    width: 100px;
    background: #FFA; 
}
<div class="wrap">
  <div class="prop">1.prop<br>float:left</div>
  <div class="main">2.main<br>float:right<br>Fairly short text</div>
  <div class="end">3.end<br>float:right</div>
</div>

我们定义了一个wrap元素和它的三个子元素prop、main、end。其中prop元素左浮动，main、end元素右浮动。它们的宽度和高度也做了一定的设计。它们的效果如下：

想象一下，当main元素中的内容过多溢出时，这几个元素的布局会是怎样的呢？由于wrap未设置overflow: hidden属性，当main元素中的内容过多溢出时，end元素会被挤出wrap，而由于我们设置的end的宽度为100px, end元素会浮动到prop元素的下面，这个时候的效果如下：

<div class="wrap">
  <div class="prop">1.prop<br>float:left</div>
    <div class="main">2.main<br>float:right<br>Call me Ishmael.  Some years ago &ndash; never mind how long precisely &ndash; having little or no money in my purse, and nothing particular to interest me on shore, I thought I would sail about a little and see the watery part of the world.  It is a way I have of driving off the spleen, and regulating the circulation.</div>
    <div class="end">3.end<br>float:right</div>
</div>

那么你可能会说，这跟多行文本溢出显示省略号有半毛钱关系吗？

显然，直接看是没什么关系的。但是如果你仔细想想，我们现在其实已经达到了某种类似的效果：当文本未溢出时显示出了end, 当文本溢出时end跑别处去了。

只是可惜了，和我们想要的效果正好相反，但恰恰说明了我们是有办法改进的。

二、 step2: 回归正题

从第一部我们看到了我们达到了和目标类似但相反的效果，但是我们仔细看看上面的代码我们可以发现，我们是能够做到预期效果的。

假设我们把main当做我们的文本区域，那么在文本内容未溢出的情况下，我们如果不想显示end元素，而当文本溢出的情况下，我们想显示我们的end元素该如何做呢？ 显然，我们只需要把end元素向右边挪300px就可以做到了。这样当文本未溢出时，end会跑到wrap的右侧，当文本溢出时，end会跑到main元素的右侧。

为了更好的显示这个效果，我们在end元素内部包裹一个新的元素叫realend，这个元素会相对于end进行绝对定位，它的left会被设置成300px。为了当main中文本溢出时使realend出现wrap内，我们这里暂时加了一个-50px的top定位。在我们改造后的代码如下： .prop, .main, .end { opacity: 0.7; } .prop { float: left; width: 100px; height: 200px; background: #FAF; } .main { float: right; width: 300px; background: #AFF; } .end { float: right; width: 100px; background: #FFA; position: relative; // 使realend相对于end定位 } .realend { position: absolute; width: 100%; top: -50px; left: 300px; background: #FAA; font-size: 13px; }

<div class="wrap">
  <div class="prop">1.prop<br>float:left</div>
  <div class="main">2.main<br>float:right<br>Fairly short text</div>
  <div class="end">
    <div class="realend">4.realend<br>position:absolute</div>
    3.end<br>float:right
  </div>
</div>

想象一下，如果吧realend元素当做我们的省略号，我们是不是已经非常非常接近达到我们的效果了？

但是先别着急，我们还有其它问题：这个end元素怎么处理呢？事实上，我们发现把它留在这里不是很友好，想把它藏起来。有很多方法可以做到这一点，我们简单列举几个：

(1) 设置end元素设置visibility：hidden, realend元素设置visibility：visible;

.end {
    float: right;
    width: 100px;
    background: #FFA;
    position: relative;
    visibility: hidden;
}
.realend {
    position: absolute;
    width: 100%;
    top: -50px;
    left: 300px;
    background: #FAA; font-size: 13px;
    visibility: visible;
}

效果如下：

(2) 设置end元素全透明，子元素不透明。但要注意，父元素只能用rgba设置透明，不能使用opacity。

.end {
    float: right;
    width: 100px;
    background-color: rgba(255,255,170, 0);
    color: rgba(255,255,170, 0);
    position: relative;
}
.realend {
    position: absolute;
    width: 100%;
    top: -50px;
    left: 300px;
    color: black;
    background: #FAA; font-size: 13px;
    opacity: 1;
}

效果：

(3) 设置end元素高度为0

.end {
    float: right;
    width: 100px;
    background: #FFA;
    position: relative;
    height: 0;
    color: rgba(0, 0, 0, 0.0);
}
.realend {
    position: absolute;
    width: 100%;
    top: -50px;
    left: 300px;
    background: #FAA; font-size: 13px;
    color: black;
}

效果同上。

实际上，我们把end中的内容去掉CSS中也不会有那么多color之类的设置了。

三、 step3: 使用relative定位

实际上我们第二步已经使用决定定位的left: 300px基本上达到了效果，而第二步的末期我们还打算把end元素给隐藏起来。仔细想想，我们所做的这么多基本上行都是多余的，因为relative定位就可以做到上面两个需求（left: 300px 和 把end元素给干掉）了啊！

回顾一下，相对定位是相对于它的正常位置进行定位的。所以我们这里可以把end元素给去掉了，给realend元素一个float: right属性。这时如果不施加position，那它就和step1中的end没什么区别。我们这里给它一个position: relative定位,这样在减少一个不必要的元素的同时，还能达到step2中的效果。代码如下：

.wrap {
    width: 400px; height: 200px;
    margin: 20px 20px 50px;
    border: 5px solid #AAA;
    line-height: 25px;
}
.prop, .main, .realend {
  opacity: 0.7;
}
.prop {
    float: left;
    width: 100px; height: 200px;
    background: #FAF;
}
.main {
    float: right;
    width: 300px;
    background: #AFF;
}
.realend {
    position: relative;
    float: right;
    width: 100px;
    top: -50px;
    left: 300px;
    background: #FAA; font-size: 13px;
}

</div>

It is a way I have of driving off the spleen, and regulating the circulation.</div>

四、 step4: 将prop元素变窄

通过前面几步，我们发现我们几乎达到预期效果了，但是我们发现prop元素放在这里太占地儿了。所以我们考虑把它变窄。这里我们给它5px。并且我们想要main元素和容器宽度一致，我们设置wrap元素宽度为300px。要想使main元素铺满整个容器，我们需要给它设置margin-left: -5px; 我们的realend元素需要加上margin-left: -100px，同时为了弥补main元素的margin-left: -5px;, realend元素需要设置一个margin-right: 5px;或padding-right: 5px;或

代码如下：

.wrap {
    width: 300px; height: 200px;
    margin: 20px 20px 50px;
    border: 5px solid #AAA;
    line-height: 25px;
}
.prop, .main, .realend {
  opacity: 0.7;
}
.prop {
    float: left;
    width: 5px; height: 200px;
    background: #FAF;
}
.main {
    float: right;
    width: 300px;
    margin-left: -5px;
    background: #AFF;
}
.realend {
    position: relative;
    float: right;
    width: 100px;
    top: -50px;
    left: 300px;
    margin-right: 5px;
    margin-left: -100px;
    background: #FAA; font-size: 13px;
}

五、自适应

在前面几步中wrap和main和realend的宽度都是定死的，这很不具有自适应性，现在我们把它们改成百分比的形式，wrap我们暂定为50%，这是为了演示。main理应定义成100%，realend任然是100px，但它的left变为100%。

代码如下：

.wrap {
    width: 50%; height: 200px;
    margin: 20px 20px 50px;
    border: 5px solid #AAA;
    line-height: 25px;
}
.prop, .main, .realend {
  opacity: 0.7;
}
.prop {
    float: left;
    width: 5px; height: 200px;
    background: #FAF;
}
.main {
    float: right;
    width: 100%;
    margin-left: -5px;
    background: #AFF;
}
.realend {
    position: relative;
    float: right;
    width: 100px;
    top: -50px;
    left: 100%;
    margin-right: 5px;
    margin-left: -100px;
    background: #FAA; font-size: 13px;
}

六、使用伪元素进行优化

我们将使用after和before伪元素对我们的结构进行优化，这样我们的代码看起来会更加的简练，这里我们还将吧wrap改个名字叫做ellipsis。我们把realend的top调整为line-height。

代码如下：

.ellipsis {
    width: 50%; height: 200px;
    margin: 20px 20px 50px;
    border: 5px solid #AAA;
    line-height: 25px;
}
.ellipsis::before, .ellipsis::after, .ellipsis > *:first-child {
  opacity: 0.7;
}
.ellipsis::before {
    content: '';
    float: left;
    width: 5px; height: 200px;
    background: #FAF;
}
.ellipsis > *:first-child {
    float: right;
    width: 100%;
    margin-left: -5px;
    background: #AFF;
}
.ellipsis::after {
    content: 'realend';
    position: relative;
    float: right;
    width: 100px;
    top: -25px;
    left: 100%;
    margin-right: 5px;
    margin-left: -100px;
    background: #FAA; font-size: 13px;
}

七、加上overflow: hidden

我们在前面为了展示，溢出的文本都没有隐藏，现在我们加上overflow: hidden属性。同时，我们前面为了演示realend的位置特意把ellipsis宽度设置为了50%，这里我们也去掉，这样ellipsis宽度就是100%了，改造后的代码如下：

.ellipsis {
    height: 200px;
    margin: 20px 20px 50px;
    border: 5px solid #AAA;
    line-height: 25px;
    overflow: hidden;
}
