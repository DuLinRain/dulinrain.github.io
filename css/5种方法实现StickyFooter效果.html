5种方法实现“Sticky Footer”效果

一、什么是“Sticky Footer”效果？

所谓的“Sticky Footer”效果是指，无论页面内容多少，页脚元素都停留在页面的底部的这种效果。

假设我们的html的结构如下：

<div class="container">
    <div class="content">
       content
    </div>
    <div class="footer">
        footer
    </div>
</div>

我们需要实现类似于下面这种效果：

内容不足时：

内容足够时：

那么如何实现这种效果呢？我们下面讲述5种实现这种效果的方法。

二、方法1： 在content上使用负下边距

这种方法适用于footer是固定高度的情形，并且需要一个额外的元素。 我们先看一下html结构：

<div class="container">
    <div class="content">
        content
        <div class="push"></div>
    </div>
    <div class="footer">
        footer
    </div>
</div>

我们的CSS如下：

html, body {
    width: 100%;
    height: 100%;

}
* {
    margin: 0;
    padding: 0;
}
.container {
    width: 100%;
    height: 100%;
}
.content {
    min-height: 100%;//注意不能设置height: 100%;
    margin-bottom: -60px;
}
.push {
    height: 60px;
}
.footer {

    text-align: center;
    background-color:aliceblue;
    height: 60px;
}

实现的效果和示例是一样的。

三、方法2： 在footer上使用负上边距

这种方法同样适用于footer是固定高度的，这种方法不需要方法1中的push元素，但是这种方法也需要一个额外的元素将content包裹起来。 而content元素需要设置padding-bottom，值为footer的高度。同时，需要设置footer适用负上边距，值为footer的高度。 其html结构如下：

<div class="container">
    <div class="content-wrapper">
        <div class="content">
                content    
        </div>
    </div>

    <div class="footer">
        footer
    </div>
</div>

css如下：

html, body {
    width: 100%;
    height: 100%;

}
* {
    margin: 0;
    padding: 0;
}
.container {
    width: 100%;
    height: 100%;
}
.content-wrapper {
    min-height: 100%;
}
.content {    
    padding-bottom: 60px;
}
.footer {    
    text-align: center;
    background-color:aliceblue;
    height: 60px;
    margin-top: -60px;
}

最终效果一致。

四、方法3： 使用calc()

这种方法不需要任何额外的元素，也是大家最容易想到的。但他同样需要footer的宽度是固定的。 html的结构就是最原始的结构：

<div class="container">
    <div class="content">
            content 
    </div>
    <div class="footer">
        footer
    </div>
</div>

css的实现如下：

html, body {
    width: 100%;
    height: 100%;

}
* {
    margin: 0;
    padding: 0;
}
.container {
    width: 100%;
    height: 100%;
}
.content {   
    min-height: calc(100% - 60px);
}
.footer {    
    text-align: center;
    background-color:aliceblue;
    height: 60px;
}

最终效果一致。

五、方法4： 使用flexbox

flex布局，非常容易实现这种效果，并且footer的高度不需要固定。 也无需额外的元素。

html结构如下：

<div class="container">
    <div class="content">
            content 
    </div>
    <div class="footer">
        footer
    </div>
</div>

css如下：

html, body {
    width: 100%;
    height: 100%;

}
* {
    margin: 0;
    padding: 0;
}
.container {
    width: 100%;
    height: 100%;
    display: flex;
    flex-direction: column;
}
.content {   
    flex: 1 0 auto;
}
.footer {    
    flex-shrink: 0;
    text-align: center;
    background-color: aliceblue;
}

最终效果一致。

六、方法5： 使用grid布局

grid layout是一种新的布局，非常容易实现这种效果，并且footer的高度不需要固定。 也无需额外的元素。只是浏览器支持的不是很好。

html结构如下：

<div class="container">
    <div class="content">
            content 
    </div>
    <div class="footer">
        footer
    </div>
</div>

css如下：

html, body {
    width: 100%;
    height: 100%;

}
* {
    margin: 0;
    padding: 0;
}
.container {
    width: 100%;
    height: 100%;
    display: grid;
    grid-template-rows: 1fr auto;
}
.content {   
    grid-row-start: 1;
    grid-row-end: 2;
}
.footer {    
    text-align: center;
    background-color:aliceblue;
    grid-row-start: 2;
    grid-row-end: 3;
}

最终效果一致。
