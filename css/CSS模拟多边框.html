CSS模拟多边框

我们都知道，CSS的border属性是用来指定边框的，但是该属性并不支持逗号分隔，也就意味这我们只能实现一个边框，使用该属性实现多个边框是不可能的。

那么要想实现多个边框效果，我们可以使用多个元素来实现多边框效果，但是这样会引入额外的、不需要的元素。所以更可取的方法是采用box-shadow或outline来模拟。 下面我们来分别看一看。

1、使用box-shadow模拟

到目前为止，我们大多数人可能已经使用box-shadow来创建阴影效果。 但是，很少人用到过它的第四个参数（称为“扩展半径”），它会使阴影变大（正值长度）或更小（负值长度）。 我们知道，一般box-shadow用出来的效果都看起来和正常的border不一样，正常的border棱角分明，而box-shadow顾名思义，看起来像阴影一样。比如：

CSS代码：

mutil-border {

width: 200px;
height: 100px;
border: 10px solid blue;
box-shadow: 5px 5px 5px 10px #58b;

}

效果：

那么如何使用box-shadow创建类似于正常的border呢？答案是正的扩展半径结合0偏移与0模糊半径。比如：

CSS代码：

#mutil-border {
    width: 200px;
    height: 100px;
    border: 10px solid blue;
    box-shadow: 0 0 0 10px #58b;
}

效果：

这并不是box-shadow最令人印象深刻的地方。boxshadow最大的好处是我们可以尽可能多的用逗号分隔创建多个box-shadow，也就进而创建多个border。 所以，我们可以很容易地在前面的例子中添加第二个deeppink颜色的border。

css代码：

#mutil-border {
    width: 200px;
    height: 100px;
    margin: 50px auto;
    border: 10px solid blue;
    box-shadow: 0 0 0 10px #58b,  0 0 0 15px deeppink;
}

效果：

唯一要记住的是box-shadow会重叠在一起，第一个是在最上面。 因此，您需要相应地调整扩展半径，比如上面那个例子，粉色的扩展半径是15px，但是它上面一个box-shadow把它盖住了10px，所以我们看到的就像是5px。

box-shadow在大多数场合都可以完美的完成多边框的任务，但是我们仍然需要注意这几点：

  1. box-shadow并不就是border。因为它不会影响布局(不占地儿)，而且对box-sizing属性不感冒。
  2. box-shadow创建的假border没法感知hover和click事件，因为默认使用的是外阴影(outset)，如果这些事件是你想关注的，你可以使用内阴影(inset)，这样你可能需要调整你的padding。

2、使用outline模拟模拟

在某些情况下，如果我们只需要两个边框，我们可以使用一个常规的边框和一个outline属性，从而得到两个边框的效果。 这也给了我们带来了边框风格的灵活性（如果我们想要第二个边框是虚线边框？），而使用box-shadow方法，我们只能模拟实线边框。

css代码：

#mutil-border {
    width: 200px;
    height: 100px;
    margin: 50px auto;
    border: 10px solid blue;
    outline: 15px solid deeppink;
}

效果：

使用outline的另外一个好处是，你可以指定它离元素的偏移值(outline-offset属性)，这个值可以是正的，也可以是负的。

CSS代码：

#mutil-border {
    width: 200px;
    height: 100px;
    margin: 50px auto;
    border: 10px solid blue;
    outline: 15px solid deeppink;
    outline-offset: 10px;
}

效果：

  1. 它只适用于两个边框的场合。因为outline也不支持逗号分隔符。

关于border你还需要注意的是，如果你给元素设置了background-color，那border的颜色是会盖住这个颜色的，因为background的起始点是border-box。要想改变这种情况可以设置background-clip属性。
