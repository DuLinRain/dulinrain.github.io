一、概述

ES6支持解构，能够很方便的从数据仓库中提取数据。解构让你使用某种模式从数据仓库中提取数据。如下：

通常，解构可以用在下面四种场合：

  1. 变量声明

    let [x] = ['a']; var [x] = ['a']; const [x] = ['a'];

  2. 赋值

    [x] = ['a']

  3. 函数参数定义

    function f([x]){...} f(['a'])

二、模式

2.1、解构模式

对于解构，等号左边的解构模式通常被称作“解构目标（destrucuring target）”，右边的数据通常被称为“解构源（destructuring source）”

解构模式通常分为三种： 1. 赋值目标（assignment target）。通常用在变量声明和函数参数定义 2. 对象模式（object pattern）。比如：{first: <>, last: <>} 3. 数组模式（array patterm）。比如：[<>, <>]
