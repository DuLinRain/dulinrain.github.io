JavaScript设计模式（结构型）之组合模式

一、概述

组合模式更是简单了，组合模式的主要思路是，不需要使用者关心这个接口所处理的是一个单个元素还是一个数组。那么这个就意味着，只需要在程序中判断一下用户使用时传递的是单个元素还是数组就可以了。
