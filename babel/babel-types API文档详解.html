babel-types API文档详解

@babel/types

这个模块包含了一些用于手动地创建AST以及检查AST类型的方法。 你在编写Babel插件时会经常用到

Install

sh npm install --save-dev @babel/types

arrowFunctionExpression

javascript t.arrowFunctionExpression(params, body, async)

另见 t.isArrowFunctionExpression(node, opts) and t.assertArrowFunctionExpression(node, opts).

别名: Scopable, Function, BlockParent, FunctionParent, Expression, Pureish

  • params: Array<LVal> (required)
  • body: BlockStatement | Expression (required)
  • async: boolean (default: false)
  • expression: boolean (default: null)
  • generator: boolean (default: false)
  • returnType: TypeAnnotation | TSTypeAnnotation | Noop (default: null)
  • typeParameters: TypeParameterDeclaration | TSTypeParameterDeclaration | Noop (default: null)

示例： 实现 (a, b)=> { a = 1 }

思路： 实现方式和普通的函数一样，需要提供params，body。 区别是不需要提供id。

// params
var params = [t.identifier("a"), t.identifier("b")]

// expression
var expression = t.assignmentExpression("=", t.identifier("a"), t.numericLiteral(1))
// expressionStatement
var expressionStatement = t.expressionStatement(expression)
// BlockStatement
var body = t.blockStatement([expressionStatement])
t.arrowFunctionExpression(params, body)

ifStatement

javascript t.ifStatement(test, consequent, alternate)

See also t.isIfStatement(node, opts) and t.assertIfStatement(node, opts).

Aliases: Statement, Conditional

  • test: Expression (required) 必须是一个合法的表达式
  • consequent: Statement (required) 必须是一个声明
  • alternate: Statement (default: null) 必须是一个声明

示例： 要实现的是下面这段代码：

if (a===1) { 
    a=1 
} else { 
    b=2 
}

思路，我们需要构造一个ifStatement，可分解为构建两个blockStatement和一个BinaryExpression。

// a===1
var binexpression = t.binaryExpression("===", t.identifier("a"), t.numericLiteral(1)) 
//{ a=1 }，第一个blockStatement
var expression1 = t.assignmentExpression("=", t.identifier("a"), t.numericLiteral(1)) // a=1
var expressionStatement1 = t.expressionStatement(expression1)
var consequent = t.blockStatement([expressionStatement1]) //块声明： {a=1}

//{ b=2 }，第二个blockStatement
